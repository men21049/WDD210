Design principles

PARC: Alignment

Youtube.com is an example of an Alignment design, because in mobile and desktop version, we can see a grid of videos displayed, there is a clear structure of alignment between each element of the wesite.

PARC: Repetition

Google Anlytics is an example of an repetition design, once we access the website in either mobile or desktop version, we can see that some elements and styles are repeated throughput the website. And there is emphasis on the different aspects goole analytics works for different websites.

White Space and Clean Design

Coursera is an example of White Spaces and clean design, The website is clean and every section of the website is focus towards just one section, when you scroll down you can see that you are always invited to register and you can clearly understand what the website looks
